#Fireplay

Fireplay es una aplicación web moderna para explorar y comprar videojuegos, desarrollada con Next.js y TypeScript. Permite buscar títulos, agregarlos a favoritos o al carrito, y simular una compra. 



#Características

- Búsqueda de videojuegos por nombre.
- Página de detalles con imágenes y valoraciones.
- Agregar juegos a favoritos y al carrito.
- Cálculo del total y simulación de compra.
- Persistencia de datos en localStorage.
- Optimización de imágenes con `next/image`.



#Tecnologías utilizadas

- Next.js 15
- TypeScript
- Tailwind CSS
- Context API (React)
- Vercel
